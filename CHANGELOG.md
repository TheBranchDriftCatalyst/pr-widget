# Changelog

All notable changes to PR Widget will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-03-13

### Added

- **app**: Native macOS menu bar app with floating NSPanel
- **auth**: GitHub token storage via macOS Keychain
- **auth**: Multi-account support with account switching
- **dashboard**: PR list grouped by repository with collapse/expand
- **dashboard**: Category sections (Needs Action, Ready to Ship, Waiting)
- **dashboard**: Urgency scoring based on age, review state, CI, conflicts, size
- **dashboard**: Label filtering with include/exclude modes
- **dashboard**: Search by PR title, repo name, branch, author
- **dashboard**: Repo drag-and-drop reordering
- **dashboard**: PR pinning to dedicated section
- **actions**: Context menu with Open in Browser, Copy URL, Copy Branch
- **actions**: Label add/remove/re-apply via context menu
- **detail**: PR detail view with synopsis card and activity feed
- **team**: Team dashboard with aging board view
- **ai**: AI-powered PR summaries and review suggestions
- **settings**: Account management, AI config, hotkey customization
- **hotkey**: Global keyboard shortcut (Cmd+Shift+Option+P)
- **theme**: Catalyst cybersynthpunk design system with glass effects
- **help**: In-app help system with 11 tips across 4 categories
